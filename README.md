# NostrImgUploader

A simple web client demonstrator for [INOSTA API backend](https://github.com/johnongit/INOSTA-Nostr-Img-Service)

## Getting started

### Clone this repo

```bash
git clone
```

### Install the deps 
```bash
yarn
```

### Setup env variable

fill host on VITE_HOST field

### launch React dev server
```bash
yarn dev
```
